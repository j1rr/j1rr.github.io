Docker Media Server Lab

Docker Cloudflare Jellyfin REST API Linux WSL2

Servidor de streaming contenerizado con Docker Compose, expuesto de forma segura mediante Cloudflare Tunnel (sin abrir puertos). Incluye integración con la API de Jellyfin para obtener metadatos y URLs de reproducción, además de automatización de verificación periódica para recuperación ante fallos. El proyecto demuestra habilidades en contenerización, redes, seguridad y operación de servicios.

Containerized streaming server with Docker Compose, securely exposed via Cloudflare Tunnel (no open ports). Includes integration with Jellyfin's API to fetch metadata and playback URLs, plus periodic health-check automation for failure recovery. The project demonstrates containerization, networking, security, and service operations skills.
